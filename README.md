Task Manager - Spring Boot OOP Project
Project Description

This project is a simple Task Management System developed using Spring Boot.
It demonstrates Object-Oriented Programming (OOP) principles and layered architecture.

Architecture
The project follows layered architecture:
Controller layer
Service layer
Repository layer
Model layer

Features (CRUD)
Create Task
Get All Tasks
Get Task By ID
Update Task
Delete Task

Data Storage
Tasks are stored in memory using a List collection.
Console Logging
All operations are printed to the console.

Technologies Used
Java
Spring Boot
REST API

How to Run
Clone the repository
Open in IntelliJ IDEA
Run Oop12Application.java
Use Postman: http://localhost:8080/tasks
